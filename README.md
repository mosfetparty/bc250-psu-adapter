# bc250-psu-adapter
BC250 adapter for proper ATX PSU PS_ON signal control. 
Enables automatic power control without adittional buttons and jumpers.


<img width="640" height="640" alt="lite_front" src="https://github.com/user-attachments/assets/8c4615e9-8f11-44e3-84f0-b3fbe80748d8" />



There are 3 versions:
1. Plug-n-play for FSP500-30AS, best, in my opinion, power supply for BC250.
2. Plug-n-play for ATX 24pin (standard PC power supply).
3. DIY universal mini adapter

you can find them on my website:  [mosfet.party](https://mosfet.party/)

installation video: https://youtube.com/shorts/l6obiqGzoOI?si=GQPnjSv2MyyRJluW
